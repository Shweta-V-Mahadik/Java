Introduction to Web Development

Web development is the process of creating websites and web applications that run on the internet. It involves designing, coding and maintaining web pages using various technologies.


Objectives of Web Development
  • Understand basic structure of HTML document
  • Learn formatting and text alignment
  • Use lists, images and hyperlinks

Core components of a Web Page

1. HTML

  • Defines structure of the page
  • Use tags like <html>, <head>, <body>
2. CSS
  • Used for styling the web page
  • Controls layout and colors
3. JavaScript
  • Adds Interactivity
  • Handles events and validations

Text Formatting Options

HTML allows text to be bold , italic, underlined and even colored to make the content creative

Special Characters

Some special symbols used in HTML are :

  • Copyright symbol : ©
  • Registered trademark : ®
  • Trademark : ™
  • Less than sign :<
  • Greater than sign :>
  • Ampersand :&
